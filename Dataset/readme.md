## Dataset

The dataset contains simulated user-level A/B testing data designed to evaluate landing page conversion performance.

It includes user demographics, engagement behavior, device information, geographic location, and conversion outcomes for both **control** and **treatment** groups.

### Key Features
- `user_id` → Unique user identifier  
- `group` → Control or Treatment group  
- `landing_page` → Landing page version shown  
- `converted` → Conversion outcome (0 = No, 1 = Yes)  
- `age` → User age  
- `gender` → User gender  
- `session_duration` → Time spent on website  
- `pages_visited` → Number of pages viewed  
- `device_type` → User device category  
- `location` → User geographic location  
- `purchase_amount` → Purchase value
